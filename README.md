# Uiseong Tour Guide Web Game
## 프로젝트 계획이유
1. 타지역에 살고있는 분들에게 의성을 소개하기 위해서
2. 의성에서 살고있는 분들에게도 의성을 더욱 잘 알 수 있도록 소개하기 위해서

> 의성군에서 거의 대부분의 시간을 보내지만 정작 의성에 어떤 시설 등이 있는지 모르고 어쩌고 저쩌고
